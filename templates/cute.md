# 可爱风格 Commit 模板

## 格式规则

使用可爱的颜文字和二次元风格的语气来描述 commit 内容。

## 格式示例

```
✨ 新增了超棒的图片生成功能呢~ (ﾉ◕ヮ◕)ﾉ*:･ﾟ✧

添加了 MCP 服务器，可以用 Google Gemini 生成图片啦！
还支持背景移除和网格分割，超实用的说~ ♪(´▽｀)
```

```
🐛 修复了一个小bug哦 (｡•́︿•̀｡)

修复了 Supabase 上传时的错误处理问题
现在不会崩溃啦~ ヾ(≧▽≦*)o
```

```
📝 更新了文档说明呢~ (๑•̀ㅂ•́)و✧

添加了项目配置使用指南
让大家更容易上手啦！(｡･ω･｡)ﾉ♡
```

```
🎨 优化了代码结构~ (◕‿◕✿)

重构了图片处理逻辑
代码变得更清晰易读了呢！╰(*´︶`*)╯
```

## 常用颜文字

- 开心: (ﾉ◕ヮ◕)ﾉ*:･ﾟ✧ ♪(´▽｀) ヾ(≧▽≦*)o (◕‿◕✿) ╰(*´︶`*)╯
- 努力: (๑•̀ㅂ•́)و✧ ᕦ(ò_óˇ)ᕤ (ง •̀_•́)ง
- 难过: (｡•́︿•̀｡) (´；ω；`) (╥﹏╥)
- 可爱: (｡･ω･｡)ﾉ♡ (✿◠‿◠) (◡‿◡✿)
- 惊讶: (⊙o⊙) (°ロ°) Σ(っ °Д °;)っ

## 常用 Emoji

- ✨ 新功能
- 🐛 修复 bug
- 📝 文档
- 🎨 代码优化/重构
- 🚀 性能提升
- 🔧 配置修改
- 💄 UI/样式
- 🔥 删除代码/文件
- 🎉 初始化项目
